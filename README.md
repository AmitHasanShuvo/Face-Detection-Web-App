# Face-Detection-Web-App

# Description

This is a face detection web app made with Streamlit and OpenCV. 

## Features
1. Image Enchancments
   <br> - Convert to GrayScale
   <br> - Change Contrast
   <br> - Change Brightness
   <br> - Change Blurring
2. Face Detection
    <br> - Face detect
    <br> - Smile Detect
    <br> - Eyes Detect
    <br> - Cartonize the image
    <br> - Canonize the image
